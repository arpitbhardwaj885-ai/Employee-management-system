# Employee Management System (Python)

## 📖 Overview
This project is a simple command-line based Employee Management System developed using Python.  
It allows users to manage employee records efficiently using a dictionary data structure.

---

## 🚀 Features
- ➕ Add Employee  
- 📋 View All Employees  
- 🔍 Search Employee by ID  
- ❌ Exit Program  

---

## 🛠️ Technologies Used
- Python
- Dictionaries (Data Structure)
- Functions
- Loops and Conditional Statements

---

## ⚙️ How It Works
The system runs using a menu-driven interface:
1. User selects an option from the menu  
2. The corresponding function is executed  
3. Employee data is stored temporarily in a dictionary  

---

## ▶️ How to Run

1. Open terminal or VS Code  
2. Navigate to project folder  
3. Run the file:
